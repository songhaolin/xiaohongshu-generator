## 💡 优化技巧

### 1. 提高互动率
```
✅ 互动引导语要具体
   "你家冰箱有什么去味妙招？评论区分享！"

❌ 互动引导语太宽泛
   "大家觉得怎么样？"
```

### 2. 提高收藏率
```

   "3个方法，每个都亲测有效"

✅ 结构清晰易保存
   分段明确，重点突出
```

```

   "明天分享XX，关注不迷路"

```

## ❓ 常见问题

### Q1: 生成的内容质量不好怎么办？
```
A:
1. 检查搜索结果质量
2. 尝试不同的关键词
3. 手动调整生成的内容
4. 告诉skill需要什么样的风格
```

### Q2: 选题会重复吗？
```
A:
- 选题库有200+主题
- 随机选择，重复概率低
- 也可以手动指定主题
- 定期更新选题库
```

### Q3: 需要什么设备？
```
A:
- 手机/电脑（必需）
- AI绘画工具账号（必需）
- 小红书账号（必需）
- 图片编辑软件（可选）
```

### Q4: 每天必须更新2条吗？
```
A:
- 建议每天2条
- 实在忙可以1条
- 保持持续更新
- 质量比数量重要
```
